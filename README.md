# Productivity Booster

> **"Get things done with style."**

A powerful, **Neo-Brutalist** productivity suite designed to keep you focused and organized. Built with vanilla web technologies, it runs entirely in your browser.

## Features

### Neo-Brutalism Design
- **Color Boost Engine**: Customize the entire app's color scheme with a built-in hue/saturation/brightness slider.
- **Bold Aesthetics**: Thick borders, hard shadows, and high contrast for a distinct look.

### Task & Life Management
- **Tasks**: Add, prioritize, and complete your daily to-dos.
- **Habits**: Track daily streaks for your recurring habits.
- **Goals**: Set numeric targets (pages read, hours coded) and track progress.

### Focus Tools
- **Pomodoro Timer**: Includes **Work (25m)**, **Short Break (5m)**, and **Long Break (15m)** modes.
- **Meditation Zone**: A dedicated section with a breathing visualizer and calm audio.
- **Quick Notes**: A sticky note area that auto-saves your thoughts instantly.

### Analytics Dashboard
- **Daily Activity**: Bar charts tracking your task completion over the last 7 days.
- **Monthly Focus**: Breakdown of your productivity distribution (Tasks vs. Pomodoros vs. Habits).

## How to Run

1.  **Clone or Download** this repository.
2.  **Open `index.html`** in any modern web browser.
3.  *Optional*: For the meditation sound to work, place a file named `meditation.mp3` in the root folder.

## Technologies Used

- **HTML5 & CSS3** (Custom Properties / CSS Variables)
- **JavaScript (ES6+)**
- **Chart.js** (for Analytics)
- **LocalStorage API** (for data persistence)
- **Google Fonts** (Courier Prime, Work Sans)
